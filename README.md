# Basic Data Exploration and Cleaning using Pandas

## Objective

The objective of this project is to learn Python basics and perform basic data exploration and cleaning using Pandas.

## Dataset Used

Dataset: Ecommerce Shopping Dataset (tshirts.csv)

## Tasks Performed

1. Loaded the CSV dataset into a Pandas DataFrame.
2. Explored the dataset using:
   - head()
   - tail()
   - shape
   - columns
   - data types
3. Identified missing values.
4. Removed duplicate records.
5. Created a derived column (popularity_score).
6. Saved the cleaned dataset as a new CSV file.

## Files Included

- Basic_Data_Exploration.ipynb
- cleaned_tshirts.csv

## Tools Used

- Python
- Pandas
- Jupyter Notebook
- GitHub

## Outcome

Successfully performed data exploration and cleaning operations and exported the cleaned dataset.
