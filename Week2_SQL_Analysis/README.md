# SQL Superstore Analysis

## Project Objective
Analyze the Superstore sales dataset using SQL by applying filtering, aggregation, sorting, and business queries.

## Tools Used
- MySQL 8.0
- MySQL Workbench
- GitHub

## Dataset
Sample Superstore Dataset (CSV)

## SQL Concepts Covered
- SELECT
- WHERE
- GROUP BY
- ORDER BY
- LIMIT
- COUNT()
- SUM()
- AVG()
- HAVING
- Aggregate Functions
- Data Validation

## Business Questions Solved
- Data Exploration
- Sales Filtering
- Category Analysis
- Region-wise Sales
- Monthly Sales Trends
- Top Customers
- Top Products
- Top Cities
- Top States
- Profit Analysis
- Duplicate Order Detection
- Missing Values Validation
